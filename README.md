This is a simple and interactive Currency Converter built using HTML, CSS, JavaScript. The application allows users to convert one currency into another using real-time exchange rates.

## Features

Real-time currency conversion

Supports multiple international currencies

Clean and responsive UI

Fast and accurate calculations

Beginner-friendly project using pure JavaScript
## Tech Stack

HTML – Structure

CSS – Styling & responsiveness

JavaScript – Logic & API integration

 ## Purpose

This project was created to practice JavaScript fundamentals, API handling, and DOM manipulation, making it a great beginner-level web development project.
